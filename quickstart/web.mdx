Anywhere you can run clientside JavaScript, you can run the Vapi web SDK.

- vanilla JS implementations with raw HTML & JavaScript
- Reactjs
- Framework-level like Next.js &

The Vapi web dashboard uses the Vapi web SDK underneath — so this guide will peel back the layers a bit more to make it more clear how you can programmatically talk to Vapi yourself from a web application.
